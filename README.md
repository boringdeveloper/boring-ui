## Technologies Used

- `pnpm`
- React with Vite
- React Aria
- TailwindCSS
- Storybook

## Development Setup

1. Clone the repo.
2. Install the dependencies using `pnpm`

```bash
pnpm i
```

3. Run the Storybook

```bash
pnpm storybook
```

This will build Tailwind and start an instance of Storybook in port [6006](http://localhost:6006/).
